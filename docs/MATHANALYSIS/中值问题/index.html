Langrange 中值定理

拉格朗日中值定理:

若f(x)f(x)f(x)在[a,b][a,b][a,b]上连续,(a,b)(a,b)(a,b)上可导,则存在一个点ξ∈(a,b)\xi \in (a,b)ξ∈(a,b),使得 f′(ξ)=f(b)−f(a)b−af'(\xi)=\frac{f(b)-f(a)}{b-a}f′(ξ)=b−af(b)−f(a)​
