函数的一致连续性

一致连续用一句话概括就是:函数图像没有那么陡,具体慢慢学着体会。

一致连续的定义:​

∀ϵ>0,∃δ(ϵ)>0,∀x,y∈I,where  ∣x−y∣<δ,then  ∣f(x)−f(y)∣<ϵ \forall \epsilon>0,\exist \delta(\epsilon) >0 ,\forall x,y\in I,where\; |x-y|<\delta, then \; |f(x)-f(y)|<\epsilon∀ϵ>0,∃δ(ϵ)>0,∀x,y∈I,where∣x−y∣<δ,then∣f(x)−f(y)∣<ϵ

第一类基本问题就是让我们使用一致连续的定义去证明某个函数的一致连续性。最关键的在于分析出δ\deltaδ的取值

现在,我们来解释一致连续和非一致连续的含义以及区别

一致连续就是说,在区间III上,当两个点充分接近(∣x−y∣<δ|x-y|< \delta∣x−y∣<δ)的时候,函数也充分接近(∣f(x)−f(y)<∣ϵ|f(x)-f(y)<| \epsilon∣f(x)−f(y)<∣ϵ)

非一致连续就是说,在区间III上,仅管两个点充分接近(∣x−y∣<δ|x-y|< \delta∣x−y∣<δ)了,函数值仍然做不到充分接近,那么就一定会大于等于某个ϵ0\epsilon_0ϵ0​

这一点就启发我们如何证明非一致连续,因为要对任意小的δ\deltaδ,我们都要找到∣x−y∣<δ|x-y|<\delta∣x−y∣<δ的xxx和yyy,这就启发我们找一组任意接近的子列xn,yn{x_n},{y_n}xn​,yn​,让子列的距离∣xn−yn∣<δ|x_n-y_n|<\delta∣xn​−yn​∣<δ(这是可以做到的),然后让∣f(xn)−f(yn)∣|f(x_n)-f(y_n)|∣f(xn​)−f(yn​)∣的值大于等于某个ϵ0\epsilon_0ϵ0​
